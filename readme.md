# 🐉 HSK 2 — Atlas Interativo (Mobile)

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

Aplicativo web interativo para estudo do **HSK 2** (Exame Oficial de Chinês Mandarim — Nível 2), **otimizado para celular** com interface em Português do Brasil.

---

## ✨ Funcionalidades

| Funcionalidade | Descrição |
|---|---|
| 🃏 **Cartões 3D** | Flashcards com animação flip para cada palavra |
| 💬 **Textos e Diálogos** | Diálogos reais do HSK Standard Course 2 |
| ✍️ **Caligrafia** | Canvas de escrita com grade e caractere guia |
| 🎯 **Quiz** | Múltipla escolha com 4 opções e pontuação |
| 🔤 **300 Palavras** | Vocabulário completo do HSK 2 com busca |
| 📱 **Mobile-First** | Bottom sheet, gestos de toque, layout adaptativo |
| ⌨️ **Atalhos** | `← →` navegar · `Espaço` virar · `1/2/3` modo |
